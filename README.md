# COLD CASE
AI-Powered Interactive Mystery Game

---

## Overview

**COLD CASE** is an experimental mystery investigation game where AI generates unique cases and suspects for every playthrough.

Players become an unexpected participant in a crime investigation — as a witness, a suspect, or simply someone who received the wrong message.

Each case is dynamically created, with AI generating characters, motives, and clues to form a solvable mystery.

---

## Core Gameplay Loop

Receive case notification → Review case file →  
Interrogate suspects → Connect clues →  
Build a theory → Accuse the culprit.

---

## Key Features

**AI-Generated Cases**  
Each investigation includes dynamically generated suspects, motives, and clues.

**Push-Based Story Entry**  
The case begins through an unexpected notification that pulls the player into the mystery.

**Free-Form Interrogation**  
Players question suspects through conversational dialogue rather than fixed dialogue options.

**Personality-Driven NPCs**  
Each suspect has unique speech patterns, motives, and emotional reactions.

**Clue Board System**  
Collected evidence can be connected visually to form investigative theories.

**Season Story Structure**  
Individual cases connect to larger seasonal narrative arcs.

COLD CASE explores how **AI-generated narrative systems** can create replayable detective gameplay where every investigation feels unique.
