# 🔍 COLD CASE

> An AI-powered interactive mystery game where AI creates unique cases and reaches out to players in real life.

---

## 📋 Overview

One day, out of nowhere, you receive a notification.

As a witness, a person of interest, or a wrong number recipient — you're suddenly pulled into a case you never asked for.

Inside an AI-generated case, you interrogate suspects, connect clues, and uncover the truth.

Every case is different. Every suspect has their own personality. Every playthrough is a brand new experience.

---

## ⚡ Key Features

### 🤖 AI-Generated Cases

AI automatically creates the entire case — background, victim, suspects, tricks, and clues.

A logic engine verifies cause-and-effect between clues to prevent plot holes.

No two cases are ever the same.

### 📲 Push-Based Immersion

AI reaches out to the player first through push notifications.

Three random intro types per case:

- **Witness** — "An incident occurred near your area last night."
- **Targeted** — "This number was found in the victim's belongings."
- **Wrong Message** — "Help me... this is the only number I remember."

### 🎙️ Free-Form Interrogation

Interrogate suspects through free conversation, not pre-set choices.

Each NPC has a unique personality, speech pattern, secrets, and emotional state.

The culprit strategically lies — but slips up when confronted with key evidence.

### 🗺️ Clue Board System

Organize collected clues in a mind-map style board.

Connect clues to build your own theory.

Accuse the suspect and enter the final interrogation.

### 🎬 Season Arc System

Individual cases (5-7 days) → Season (4-5 cases) → Season arc (interconnected storyline).

Unsolved cases may become foreshadowing for future cases.

## 📱 Screen Structure

| Screen | Description |
|--------|-------------|
| Lock Screen | Push notification slides in to pull player into the case |
| Case File | Case overview, victim info, timeline, real-time alerts |
| Suspects | Suspect profiles with motive, alibi, trust gauge |
| Interrogation | Messenger-style chat UI with AI-driven suspect responses |
| Clue Board | Mind-map visualization + collected/locked clue list |
| Accusation | Final suspect selection and last interrogation trigger |

## 🎮 How to Play the Demo

1. Wait for the push notification on the lock screen
2. Tap the notification to enter the case
3. Read the case file and timeline
4. Go to Suspects tab and interrogate each one
5. Check the Clue Board to connect evidence
6. Go to Accusation tab and make your final call
